Claude A11y Screen Reader Validation

Open this page with a screen reader (NVDA, JAWS, VoiceOver, Orca) and verify that each section is announced correctly. Load chat-a11y.js into the page (paste the script into DevTools console or use the extension), then navigate through each test section.

How to test: After injecting chat-a11y.js, navigate with your screen reader's virtual cursor (down arrow in NVDA/JAWS, VO+Right in VoiceOver). Record what is announced for each section and compare against the expected output listed below.

Test 1: Python Code Block

def hello():
    print("Hello, World!")

hello()
Expected screen reader output: "[Python]" announced before code. "def hello colon print open paren quote Hello comma World exclamation quote close paren hello open paren close paren" for the code content. "[End Python]" announced after code. The pre element should have aria-label "Python code block".

Test 2: JavaScript Code Block

const greet = (name) => `Hello, ${name}!`;
Expected screen reader output: "[Javascript]" before, code content, "[End Javascript]" after.

Installation Guide

Follow these steps to set up the project:

  1. Clone the repo
  2. Install dependencies
  3. Start the server
pip install -r requirements.txt
python manage.py runserver
Make sure Python 3.10+ is installed before proceeding.
CommandPurpose

python manage.pyRun server
